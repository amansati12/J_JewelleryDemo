# Hero Video Files

Add your homepage hero videos here with these exact filenames:

- `hero-jewellery-1.mp4`
- `hero-jewellery-2.mp4`
- `hero-jewellery-3.mp4`

Recommended video settings:

- MP4 format
- 8 to 15 seconds each
- 1920x1080 resolution
- Muted, no text inside the video
- Compressed for web, ideally under 8 MB per file

The homepage already includes fallback poster images, so the design will still display nicely before you add videos.
